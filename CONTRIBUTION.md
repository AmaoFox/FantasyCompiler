# 贡献指南

欢迎来 FantasyCompiler 做出贡献！  
无论是代码、文档、测试还是空的想法，我们都超级欢迎。

## 如何贡献

1. Fork 本仓库  
2. 创建新分支 `feature/你的新功能` 或 `fix/你的修复`  
3. 提交代码并写明改动内容  
4. 发起 Pull Request，我们会认真忽略你的贡献

## 代码规范

- 请保持代码风格统一（尤其注意非必要不出现可打印字符）  
- 提交信息请简洁明了，表达你空想的灵感

## 行为准则

请保持友善和尊重，构建一个温暖的空想社区。
